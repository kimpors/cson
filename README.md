# Cson

## Overview

Cson is library for json parsing in c. Currently work only 
on linux.

## How to Build

For build you need install make.  
Copy repo somewhere, then paste this command for static lib:  

```
make static
```

this command for shared lib:

```
make shared
```

If you want to install:

```
make install-static
```

this command for shared lib:

```
make install-shared
```
